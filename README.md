## FMCG Demand Forecasting & Promotion Impact Analysis

This project demonstrates an end-to-end data science solution for demand forecasting in an FMCG retail context.

### Key Features
- SKU and store-level sales analysis
- Baseline demand forecasting using moving averages
- Promotion impact visualization
- Business-oriented evaluation using MAPE

### Business Value
The solution supports inventory planning by identifying expected demand and promotion-driven distortions in historical sales data.

App is available at https://fmcg-sales-forecasting.streamlit.app/
